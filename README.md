This project is an Employee Management System built using ASP.NET Core MVC and ASP.NET Core Web API.  
The MVC application consumes data from the Web API and displays employee information.

 setup instructions

1. Clone the repository
2. Open solution in Visual Studio
3. Update connection string in `appsettings.json`
4. Run database scripts to create required table.
5. Press Run to start application


Url's

- MVC Employee List  
  `/Employee/Index`

- Web API Endpoint  
  `/api/employees`

- Razor View Consuming API  
  `/Employee/ApiEmployeeList`
